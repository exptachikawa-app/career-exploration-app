# キャリア探索ツール

## 📋 プロジェクト概要

**就労選択支援事業所EXP立川**が提供する、就労経験のない利用者や特別支援学校卒（もしくは在学中）の方を対象とした、興味関心から職業を探索できるWebアプリケーションです。

ホランドのRIASEC理論（六角形モデル）に基づき、利用者の興味・関心を6つの領域に分類し、それぞれに対応する職業を提示します。

---

## 🎯 主な機能

### ✅ 完成済み機能

1. **ホーム画面**
   - アプリの説明と開始ボタン
   - アクセシビリティを考慮した設計（漢字ルビ、平易な表現）

2. **障害者雇用確認画面**
   - 利用者が障害者雇用を検討しているかを確認
   - 結果画面で表示する情報を最適化

3. **段階的質問システム**
   - **第1段階（抽象的質問）**: 6つの興味領域について各1問
   - **第2段階（具体的質問）**: 「わからない」と回答した領域のみ、2-3問の詳細質問
   - 視覚的な選択肢（絵文字付きボタン）
   - 進捗バー表示

4. **結果表示画面**
   - **RIASECレーダーチャート**: 6つの興味領域を視覚化
   - **TOP3興味領域**: 上位3つの興味を強調表示
   - **職業リスト**: 各領域に対応する5職種（計15職種）を表示
   - 各職業の説明文
   - 障害者雇用検討者向けの追加情報
   - 次のステップ提案

5. **PDF出力機能**
   - 結果画面をPDFでダウンロード可能
   - 支援者との共有や記録保存に活用

6. **レスポンシブデザイン**
   - PC、タブレット、スマートフォンに完全対応
   - EXP立川のロゴを使用した統一感のあるデザイン

## 📊 RIASEC理論に基づく6つの興味領域

| 領域コード | 日本語名 | 説明 |
|----------|---------|------|
| R (Realistic) | ものづくり・体を動かす | 手や体を使って、ものを作ったり、動かしたりすることが好き |
| I (Investigative) | 調べる・考える | 新しいことを学んだり、問題を解決したりすることが好き |
| A (Artistic) | つくる・表現する | 絵を描いたり、音楽を作ったり、自分を表現することが好き |
| S (Social) | 人を助ける・支える | 人と関わったり、人の役に立ったりすることが好き |
| E (Enterprising) | リーダー・企画する | 人をまとめたり、新しいことを始めたりすることが好き |
| C (Conventional) | 正確に・きちんと進める | きちんと整理したり、ルール通りに進めたりすることが好き |

## 🚀 使用方法

### 開発環境での動作確認

1. **ファイルを開く**
   ```
   index.htmlをブラウザで開く
   ```

2. **アプリの流れ**
   - ホーム画面で「はじめる」をクリック
   - 障害者雇用を検討しているかを選択
   - 第1段階の質問（6問）に回答
   - 「わからない」と回答した領域がある場合、第2段階の詳細質問に回答
   - 結果画面で興味・関心マップと職業リストを確認
   - 必要に応じてPDFをダウンロード

## 📍 地域性の考慮

職業データベースは**立川周辺、多摩エリア、東京近郊**の求人状況を考慮して構築されています。

### 地域特性を反映した職種例

- **立川周辺の企業集積**: 事務スタッフ、経理スタッフの求人が豊富
- **多摩地区の特性**: 農業・園芸スタッフ、福祉施設（介護・福祉スタッフ）
- **物流拠点**: 倉庫管理スタッフ、配送・物流スタッフ
- **医療機関**: 医療事務スタッフ

## 🔒 倫理的配慮と法令遵守

### 位置づけ

このアプリは**導入的な自己理解ツール**であり、専門的な心理検査ではありません。結果画面には以下の注意書きを明記しています：

- 「この結果は、あなたの今の興味・関心を示すものです。仕事を決めつけるものではありません。」
- 「色々な経験を通して、興味は変わることがあります。」

### 個人情報保護

- **データ保存なし**: 利用者の回答データはサーバーに保存されません
- **PDF出力のみ**: 利用者が自分で結果をダウンロードして管理
- **同意取得不要**: サーバー保存がないため、個人情報保護法上の複雑な同意取得プロセスは不要

### 障害者権利条約の観点

- **自己決定の尊重**: 職業を決めつけない表現、「向いているかもしれない」という柔らかい表現
- **合理的配慮**: 漢字ルビ、視覚的選択肢、支援者同席を想定した設計

## 🎓 理論的根拠

このアプリは、ジョン・L・ホランド（John L. Holland）が提唱した**RIASEC理論**（職業興味の六角形モデル）に基づいています。

### ホランドの六角形モデルとは

個人の職業興味を6つの類型（RIASEC）に分類し、それぞれの類型に対応する職業環境を示す理論です。広く職業カウンセリングや進路指導で活用されています。

### 参考文献

- Holland, J. L. (1997). *Making Vocational Choices: A Theory of Vocational Personalities and Work Environments*. Psychological Assessment Resources.
- VPI職業興味検査（日本版）

## 💡 支援者向けガイド

### アプリの活用方法

1. **初回面談での導入ツール**
   - 利用者の興味・関心を把握する第一歩として活用
   - 結果をもとに対話を深める

2. **職業探索のきっかけ作り**
   - 「知らなかった職業」への気づきを促す
   - 興味の幅を広げる

3. **支援計画の立案**
   - 結果をもとに、職場見学や職業体験の計画を立てる
   - 個別支援計画の参考資料として活用

4. **家族との共有**
   - PDF出力機能で結果を家族と共有
   - 進路について話し合うきっかけに

### 注意点

- **結果は絶対ではない**: あくまで今の時点での興味・関心であることを強調
- **経験の重要性**: 実際にやってみることで興味が変わることを伝える
- **強みの発見**: スコアが低い領域も否定せず、多様な興味を認める
